# 🌍 Visa CRM Management System

A modern and powerful Visa CRM Management System built with **React JS + Vite**.  
This project is designed for managing visa clients, targets, user activities, notifications, and performance analytics with a beautiful and responsive admin dashboard.

---

# 🚀 Project Overview

This CRM system helps agencies manage:

- Visa Applications
- Client Information
- User Management
- Monthly Targets
- Performance Analytics
- Notifications
- SMS Communication

The system includes a professional dashboard with modern UI, dark/light mode, filtering system, and real-time management features.
